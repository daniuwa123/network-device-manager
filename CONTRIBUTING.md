## 行为准则

### 我们的承诺

为了营造一个开放和热烈的社区，我们作为贡献者和维护者，致力于使参与本项目和社区的每个人都获得无骚扰的体验，无论其年龄、体型、身体障碍情况、种族、性别认同和表达、经验水平、教育程度、社会经济地位、国籍、外表、种族、宗教或性身份和定向如何。

### 我们的标准

贡献者应该采用的行为包括：

- 使用欢迎和包容的语言
- 尊重不同的观点和经验
- 接受有建设性的批评
- 关注社区利益
- 对其他社区成员表示同情

不可接受的行为包括：

- 涉及性骚扰的行为和性用语
- 发表冒犯性评论，人身攻击或政治攻击
- 公开或私下骚扰
- 未经明确同意发布他人私人信息
- 其他在职业环境中可能被合理认为不适当的行为

## 报告问题

### 报告Bug之前

- 检查 README 和文档
- 搜索现有的 Issues
- 查看常见问题解答

### 如何报告Bug

使用 Issue 模板填写以下信息：

```
## 问题描述
清晰简洁地描述问题是什么

## 复现步骤
1. 打开应用
2. 执行操作...
3. 看到错误

## 预期行为
应该发生什么

## 实际行为
实际发生了什么

## 环境信息
- OS: Windows 10, Ubuntu 20.04, macOS 11, etc.
- 应用版本: 2.8.6.5
- Python版本: 3.8, 3.9, 3.10, 3.11
- 设备厂商: Huawei, ZTE, Dawning, etc.

## 附加信息
- 错误日志（log/app.log）
- 截图或录屏
- 其他相关信息
```

## 代码贡献

### 开发环境设置

1. **Fork 项目**
   ```bash
   点击GitHub右上角的Fork按钮
   ```

2. **克隆仓库**
   ```bash
   git clone https://github.com/your-username/network-device-manager.git
   cd network-device-manager
   ```

3. **创建虚拟环境**
   ```bash
   python -m venv venv
   # Windows
   venv\Scripts\activate
   # Linux/macOS
   source venv/bin/activate
   ```

4. **安装依赖**
   ```bash
   pip install -r requirements.txt
   pip install -r requirements-dev.txt
   ```

5. **创建特性分支**
   ```bash
   git checkout -b feature/your-feature-name
   ```

### 代码风格

- **Python**: 遵循 PEP 8 标准
- **命名**: 使用清晰有意义的名称
- **注释**: 用中文或英文，保持一致
- **文档字符串**: 为主要函数添加docstring

### 测试

- 添加新功能时，编写相应的测试
- 运行现有测试，确保没有破坏
- 测试覆盖率应该≥80%

```bash
# 运行单元测试
python -m pytest tests/ -v

# 运行覆盖率检查
pytest --cov=src tests/
```

### 提交代码

1. **提交前检查**
   ```bash
   # 检查代码风格
   flake8 .
   
   # 检查类型
   mypy .
   
   # 运行测试
   pytest
   ```

2. **提交信息**
   ```
   简短的改动说明（50字以内）
   
   - 详细说明1
   - 详细说明2
   
   Fixes #123
   ```

3. **提交代码**
   ```bash
   git add .
   git commit -m "简短描述"
   git push origin feature/your-feature-name
   ```

4. **创建Pull Request**
   - 提供清晰的PR描述
   - 链接相关的Issue (#123)
   - 包含截图或演示（如适用）

### Pull Request 检查清单

- [ ] 测试通过
- [ ] 代码风格符合要求
- [ ] 添加了相关测试
- [ ] 更新了文档
- [ ] 更新了CHANGELOG
- [ ] 提供了清晰的PR描述

### 代码审查

- 维护者会在1周内进行审查
- 可能需要进行修改
- 一旦通过，会被合并到主分支
