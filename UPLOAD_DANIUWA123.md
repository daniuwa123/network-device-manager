## 🚀 3 步快速上传

### 步骤1：创建仓库（2分钟）

1. 打开：https://github.com/new
2. 填写信息：
   ```
   Repository name: network-device-manager
   Description: Enterprise-grade network device management tool
   Public: ✓ 选中
   ```
3. **重要**：不要初始化任何文件（README、License、gitignore）
4. 点击 "Create repository"

### 步骤2：复制命令（1分钟）

仓库创建成功后，GitHub 会显示以下命令，复制这三行：

```bash
git remote add origin https://github.com/daniuwa123/network-device-manager.git
git branch -M main
git push -u origin main
```

1. 打开 PowerShell
