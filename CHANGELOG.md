## 升级说明

### 从2.8.6.3升级到2.8.6.5

1. **备份配置**
   ```
   备份现有的device_config.json和terminal_data.json
   ```

2. **停止应用**
   ```
   完全关闭现有版本
   ```

3. **安装新版本**
   ```
   下载并运行新版本安装程序
   或直接替换exe文件
   ```

4. **验证升级**
   ```
   启动新版本，检查设备列表是否完整
   测试基本功能（连接、命令执行、备份等）
   ```

### 重大变更

- v2.8.6.5: 终端排序方式改变，旧配置会自动转换
- v2.8.6.3: VLAN配置格式优化，兼容旧版本
- v2.8.6.2: SNMP监控功能API有所调整

### 已知问题

- 某些老型号的华为交换机可能存在兼容性问题
- 网络延迟大于500ms时，可能出现命令超时
- 中文设备名称过长可能导致显示错误
